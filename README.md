NYC Citi Bike data 
-----------------

![gif](output.gif)

This repo contains the jupyter notebook that do EDA for NYC citi bike data from Jan 2016 to June and make folium maps. 

## pre-requisite:
- `brew install ffmpeg`
- `brew install phantomjs`

## requirements:
- python 3.x
- pandas
- numpy
- folium
- selenium
